# Sample: /Pandoc

## Basic Usage

```
/Pandoc Render the design doc to PDF
```

## With Explicit Path

```
/Pandoc /path/to/document.md
```

## Keep Frontmatter

```
/Pandoc /path/to/document.md --strip-frontmatter=false
```

## Specific Engine

```
/Pandoc Render the research findings --engine=xelatex
```

## Example Output

```
Rendered 10 pages (130KB) to Payments 2030 Design Document.pdf
```
